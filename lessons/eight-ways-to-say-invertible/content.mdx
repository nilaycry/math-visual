the invertible matrix theorem says that for a square n×n matrix A, a long list of
conditions are all equivalent. if any one of them is true, all of them are true.
if any one is false, all of them are false. most courses present this as a list
to memorize. it isn't — each equivalence has a reason, and seeing the reasons
makes the whole thing feel inevitable.

### the eight statements

[TODO: list them cleanly.
1. A is invertible
2. Ax = 0 has only the trivial solution
3. the columns of A are linearly independent
4. the columns of A span Rⁿ
5. det(A) ≠ 0
6. 0 is not an eigenvalue of A
7. the rank of A is n
8. the null space of A contains only the zero vector

these are the core eight. there are more but start here.]

### why they're all the same

[TODO: the through line is that all eight are ways of saying "A doesn't squash
any dimension." a non-invertible matrix collapses space — it takes Rⁿ and maps
it into something lower dimensional. every condition on the list is detecting
that collapse from a different angle.

go through each and show how it connects to the "collapsing" picture:
- null space being nontrivial: there's a direction that gets sent to zero
- columns being dependent: one column is redundant, so the span is smaller
- det being zero: the volume scaling factor is zero, so space got flattened
- zero eigenvalue: there's a direction that gets stretched by factor zero
- rank less than n: the output doesn't fill Rⁿ]
