### propositions and open sentences

**1.** For each sentence below, decide whether it is a proposition. If it is, say whether
it is true or false. If it is not, say what extra information is missing.

a. $7$ is prime.

b. $x + 1 = 3$.

c. There exists an integer $n$ such that $n^2 = 2$.

d. If $n$ is even, then $n+1$ is odd.

**2.** Rewrite each open sentence as a proposition by adding either a universal or
existential quantifier.

a. $x^2 \ge 0$

b. $n$ is divisible by $3$

c. $f$ is injective

### following the hypothesis

**3.** Identify the hypothesis and conclusion in each implication.

a. If $A \subseteq B$ and $x \in A$, then $x \in B$.

b. If $n$ is odd, then $n^2$ is odd.

c. If a function $f$ is bijective, then $f$ has an inverse.

**4.** Read the following attempted proof and say exactly where the logic breaks.

> Claim: If $n^2$ is even, then $n$ is even.  
> Attempted proof: Assume $n$ is even. Then $n = 2k$ for some integer $k$. So
> $n^2 = 4k^2 = 2(2k^2)$, which is even. Therefore if $n^2$ is even, then $n$ is even.

<Solution>
The computation is correct, but it proves the wrong implication. The argument starts by
assuming $n$ is even and concludes that $n^2$ is even, so it proves "if $n$ is even, then
$n^2$ is even." It never starts from the hypothesis "$n^2$ is even," so it does not prove
the stated claim.
</Solution>
