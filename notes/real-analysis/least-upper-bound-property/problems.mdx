### warmup

**1.** For each set below, decide whether it is bounded above in $\mathbb{R}$. If it is,
give one upper bound.

a. $(2,5)$

b. $\{1 - \frac{1}{n} : n \in \mathbb{N}\}$

c. $\mathbb{N}$

**2.** For each bounded set below, identify its supremum and say whether the supremum is
also a maximum.

a. $(0,1)$

b. $[0,1]$

c. $\left\{\frac{n}{n+1} : n \in \mathbb{N}\right\}$

### practice

**3.** Let $S = \{x \in \mathbb{R} : x < 3\}$. Show that $3$ is an upper bound of $S$.
Then show that any number smaller than $3$ fails to be an upper bound.

**4.** Let $S = \left\{2 - \frac{1}{n} : n \in \mathbb{N}\right\}$. Prove that $\sup S = 2$.

### challenge

**5.** Give an example of a nonempty set of rational numbers that is bounded above in
$\mathbb{Q}$ but has no least upper bound in $\mathbb{Q}$. Explain why this does not
contradict the least upper bound property of $\mathbb{R}$.

<Solution>
One standard example is

$$
S = \{q \in \mathbb{Q} : q^2 < 2\}.
$$

This set is nonempty and bounded above in $\mathbb{Q}$, for example by $2$. In
$\mathbb{R}$ its least upper bound is $\sqrt{2}$, but $\sqrt{2} \notin \mathbb{Q}$.
So there is no rational number that is the least upper bound of $S$ inside $\mathbb{Q}$.

This does not contradict the least upper bound property of $\mathbb{R}$ because that
property is a statement about subsets of $\mathbb{R}$ having suprema in $\mathbb{R}$.
It does not claim that every bounded subset of $\mathbb{Q}$ has a supremum in $\mathbb{Q}$.
</Solution>
