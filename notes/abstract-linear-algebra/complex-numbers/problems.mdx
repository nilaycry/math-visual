### definitions

**1.** Explain precisely in what sense $\mathbb{C}$ is the same as $\mathbb{R}^2$, and in what sense it is not.

<Solution>

As a set, $\mathbb{C}$ can be identified with $\mathbb{R}^2$ by $a + bi \leftrightarrow (a,b)$. Addition also agrees with coordinatewise addition under this identification.

The difference is multiplication. $\mathbb{C}$ has a multiplication rule $(a + bi)(c + di) = (ac - bd) + (ad + bc)i$ that makes every nonzero complex number invertible. Ordinary $\mathbb{R}^2$ does not come with that field structure by default.

</Solution>

**2.** In the textbook's notation, what does $\mathbb{F}$ mean? Why is this notation useful?

<Solution>

$\mathbb{F}$ means either $\mathbb{R}$ or $\mathbb{C}$. It is useful because many definitions and proofs work the same way for real and complex scalars. Stating them over $\mathbb{F}$ avoids proving the same result twice.

</Solution>

### computations with structure

**3.** Re-derive the formula for $(a + bi)(c + di)$ using only distributivity and $i^2 = -1$.

<Solution>

Using distributivity,

$$
(a + bi)(c + di) = ac + adi + bci + bd i^2.
$$

Since $i^2 = -1$, this becomes

$$
ac + adi + bci - bd = (ac - bd) + (ad + bc)i.
$$

</Solution>

**4.** Find the multiplicative inverse of $3 - 2i$.

<Solution>

Multiply by the conjugate:

$$
\frac{1}{3 - 2i} = \frac{3 + 2i}{(3 - 2i)(3 + 2i)} = \frac{3 + 2i}{9 + 4} = \frac{3 + 2i}{13}.
$$

So

$$
\frac{1}{3 - 2i} = \frac{3}{13} + \frac{2}{13}i.
$$

</Solution>

**5.** Show that $(-1 + \sqrt{3}i)/2$ is a cube root of $1$.

<Solution>

Let

$$
z = \frac{-1 + \sqrt{3}i}{2}.
$$

First compute

$$
z^2 = \frac{(-1 + \sqrt{3}i)^2}{4} = \frac{1 - 2\sqrt{3}i + 3i^2}{4} = \frac{-2 - 2\sqrt{3}i}{4} = \frac{-1 - \sqrt{3}i}{2}.
$$

Then

$$
z^3 = z^2z = \frac{(-1 - \sqrt{3}i)(-1 + \sqrt{3}i)}{4}.
$$

The numerator is a conjugate product:

$$
(-1)^2 + (\sqrt{3})^2 = 1 + 3 = 4.
$$

Thus $z^3 = 1$.

</Solution>

### proof practice

**6.** Prove that complex addition is commutative using the definition of addition in $\mathbb{C}$.

<Solution>

Let $\alpha = a + bi$ and $\beta = c + di$, where $a,b,c,d \in \mathbb{R}$. Then

$$
\alpha + \beta = (a + c) + (b + d)i.
$$

Because real addition is commutative, $a + c = c + a$ and $b + d = d + b$. Hence

$$
(a + c) + (b + d)i = (c + a) + (d + b)i = \beta + \alpha.
$$

So $\alpha + \beta = \beta + \alpha$.

</Solution>

**7.** In the proof that complex multiplication is commutative, which properties of $\mathbb{R}$ are being used?

<Solution>

Write $\alpha = a + bi$ and $\beta = c + di$. Then

$$
\alpha\beta = (ac - bd) + (ad + bc)i
$$

and

$$
\beta\alpha = (ca - db) + (cb + da)i.
$$

To show these are equal, we use commutativity of real multiplication: $ac = ca$, $bd = db$, $ad = da$, and $bc = cb$. We also use commutativity of real addition to rewrite $ad + bc$ as $cb + da$.

So the complex theorem is proved by reducing it to already-known real-number properties.

</Solution>

**8.** Prove that every nonzero complex number has a multiplicative inverse.

<Solution>

Let $z = a + bi \in \mathbb{C}$ and suppose $z \neq 0$. Then $a$ and $b$ are not both zero, so $a^2 + b^2 > 0$.

Define

$$
w = \frac{a - bi}{a^2 + b^2}.
$$

Then

$$
zw = (a + bi)\frac{a - bi}{a^2 + b^2} = \frac{(a + bi)(a - bi)}{a^2 + b^2}.
$$

But

$$
(a + bi)(a - bi) = a^2 + b^2.
$$

Therefore $zw = 1$. So $w$ is a multiplicative inverse for $z$.

</Solution>

**9.** Why does the previous proof fail for $z = 0$?

<Solution>

If $z = 0$, then $z = 0 + 0i$, so $a = 0$ and $b = 0$. The formula for the inverse would require division by

$$
a^2 + b^2 = 0.
$$

That is not allowed. More conceptually, no number $w$ can satisfy $0w = 1$.

</Solution>
