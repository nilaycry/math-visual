## What the course is trying to do

The introduction frames Math 314 as a transition from mostly algorithmic math to
theoretical math. The course is not mainly about one content area. It is about
learning how to make precise statements, read definitions, and justify claims with proofs.

That means the notes in this repo should keep returning to the same underlying skill:
turning informal mathematical language into something exact enough to prove.

## Chapter roadmap

### Orientation

- introduction
- common notation

Use for companion notes on:

- what changes in a proof-based class
- how to read hypotheses and conclusions
- notation the course will assume silently

### 1. Logic and the language of proofs

- what a proof is
- truth tables and logical connectives
- negation
- basic proof techniques
- quantifiers
- negating quantified statements

This should drive the earliest Math 314 notebook sequence.

### 2. Basic set theory

- what sets are
- inclusions and set operations
- set identities
- the link between logic and set operations
- set-builder notation
- cartesian products
- indexed sets
- power sets

This is the first place where "unpack the definition" subset proofs should become routine.

### 3. Functions

- what a function is
- domains and codomains
- injective, surjective, bijective
- composition
- inverse functions
- viewing subsets as functions into `{0,1}`

This is a strong candidate area for later visual lessons, especially injective and surjective maps.

### 4. Counting

- how to think about counting
- pigeonhole principle
- binomial formula

Treat this as proof-based counting, not contest-style trick collecting.

### 5. Modular arithmetic

- divisibility
- division algorithm
- modular arithmetic
- Euclidean algorithm
- Bezout identity
- Fermat's little theorem

Good place for selective visuals if congruence classes or modular patterns benefit from interaction.

### 6. Equations and inequalities

- what solving an equation means
- linear systems
- nonlinear equations

This is less central to the proof-transition identity of the course, so it may need shorter coverage unless the class emphasizes it heavily.

### 7. A taste of group theory

- what a group is
- group axioms
- basic theorems
- homomorphisms

Use this as a first look at abstract structure, not as a full algebra sequence.

### 8. Relations

- what a relation is
- basic properties
- relation versus function
- transpose of a relation
- relation to invertible functions

This should lead directly into equivalence relations and quotient sets.

### 9. Quotient sets and induced functions

- quotient sets
- equivalence relations
- partitions
- well-definition
- induced functions

This is one of the most conceptually important late-middle units. Spend time on well-definition.

### 10. Induction

- what induction is
- basic induction
- strong induction
- well-ordering principle

This deserves its own note cluster, not one overloaded note.

### 11. Cardinality

- what cardinality means
- countable sets
- uncountable sets

This is a capstone unit in the course arc. It should feel like proof-reading maturity has increased by the time the sequence gets here.

## Suggested notebook order in this repo

Use the textbook as the base, but keep the note order pedagogically cleaner than the raw chapter order when needed.

Recommended early sequence:

1. how to read this kind of course
2. propositions and the shape of a proof
3. truth tables and what connectives really say
4. quantifiers and how to negate them
5. direct proof, contrapositive, contradiction
6. sets and subset proofs
7. set operations and set-builder notation
8. functions, domains, and codomains
9. injective, surjective, bijective
10. relations
11. equivalence relations and quotient sets
12. induction

Then continue into counting, modular arithmetic, the group-theory taste, and cardinality.
